# Two Pointers

Use two indices to traverse an array/string (often from both ends, or one pointer for a window and one to scan).

Common scenarios:
- Sorted arrays / arrays with monotonic properties
- Removing/compacting elements
- Detecting pairs/triplets with constraints
- Partitioning around a pivot

Checklist:
- Define pointer movement rules clearly
- Avoid off-by-one errors
- Handle duplicates/edge cases
