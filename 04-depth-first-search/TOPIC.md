# Depth-First Search (DFS)

Explore nodes by going deep first using a stack/recursion.

Common scenarios:
- Traversal of trees/graphs
- Path existence (reachability)
- Backtracking across decision trees
- Component counting / connected regions

Checklist:
- Mark visited to prevent cycles
- Manage recursion depth (iterative when needed)
- For grids: boundaries + directions + visited
