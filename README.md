# Pattern-Wise-Preparation-Question-On-Leetcode-

## How to use
- Pick a topic folder (patterns 1–12).
- Open `QUESTIONS.md` and start solving problems.
- After each solve, log your approach + learnings in `ATTEMPT_LOG.md`.
- Write/keep your solution notes in `SOLUTIONS.md` when ready.
