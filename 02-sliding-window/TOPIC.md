# Sliding Window

Maintain a contiguous window over an array/string and expand/shrink it while tracking state (sum, count, frequency).

Common scenarios:
- Subarray/substring with constraints (at most/at least/exactly)
- Longest/shortest window lengths
- Frequency/uniqueness constraints

Checklist:
- Identify window validity condition
- Decide expand vs shrink strategy
- Prefer O(n) using two pointers
